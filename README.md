# 小默源 - TVBox接口源

## 简介

小默源是一个基于 TVBox 的视频源配置，提供高清影视内容。

## 接口地址

```
https://raw.githubusercontent.com/zxl7725103/xiaomo-source/main/xiaomo.json
```

或使用 CDN 加速地址：

```
https://cdn.jsdelivr.net/gh/zxl7725103/xiaomo-source@main/xiaomo.json
```

## 使用方法

1. 下载并安装 TVBox 应用
2. 打开应用，进入设置
3. 在"接口地址"中输入上述接口地址
4. 点击"确定"保存配置
5. 返回首页即可使用

## 包含站点
