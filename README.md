# 小默源 - TVBox 接口源

## 简介

小默源是一个 TVBox 接口源仓库，提供精简版和聚合版两种接口配置。

## 使用方法

### 影视仓

1. 打开影视仓应用
2. 进入设置 → 接口配置
3. 在接口地址中输入上述任一地址
4. 保存后即可使用

### TVBox

1. 打开 TVBox 应用
2. 进入设置 → 配置地址
3. 输入接口地址并保存
4. 返回首页即可使用
